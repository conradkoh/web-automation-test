# Web Automation Test
This project was started from the [BrowserBase Stagehand](https://github.com/browserbase/stagehand) project.
## Installation
### Run the install steps
```
pnpm install
pnpm exec playwright install
```

### Setup .env.local
```
OPENAI_API_KEY=<key>
```

### Run the main script
```
pnpm run main
```
